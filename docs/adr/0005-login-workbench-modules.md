# ADR 0005：登录后先进入核对工作台

## 状态

已接受

## 背景

当前登录后直接进入达人列表，普通用户还会自动进入自己的视频数据。后续会增加费用核对等业务模块，直接进入某个数据页面不利于扩展入口。

## 决策

- 登录成功后统一进入前端工作台，不新增后端批次模型。
- 工作台展示两个卡片：“本期数据核对”和“本期费用核对”。
- 数据核对卡片进入现有数据页面：管理员进入达人列表，普通用户进入自己的视频数据。
- 费用核对卡片暂不请求后端，点击后提示“功能正在开发中”。
- 数据页面和视频页面提供返回工作台入口。

## 取舍

本阶段只做前端入口占位，不改变数据库、导入逻辑、权限模型和数据查询。后续真正引入“本期/批次”时，再将工作台卡片绑定到批次接口。
